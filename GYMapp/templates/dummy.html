b) Directed Graph

In a directed graph, Kruskal's algorithm doesn't directly apply to find a minimum spanning tree. Instead, we typically look for a minimum spanning arborescence.
However, if we aim to find the minimum set of edges that connect all vertices, we can still use a modified approach.
